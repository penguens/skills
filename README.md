# penguens skills

custom agent skills for claude, built on the [agent skills spec](https://agentskills.io/specification).

## creating a skill

each skill is a folder with a `SKILL.md` file. the frontmatter requires `name` and `description` the description controls when claude loads the skill.

use the [template](./template/SKILL.md) to start.
